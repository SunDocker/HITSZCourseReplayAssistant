# 哈工大（深圳）课程回放平台助手

## 应用场景

[哈工大（深圳）课程回放平台]()的视频播放功能并不是很友好，虽然有暂停、倍速、全屏、退出全屏等功能，但**操作极其不方便**。

所以写了这么一个**非常简陋与不规范**的插件，将常用

>   或许在未来的某一天，我会将这个插件规范化
